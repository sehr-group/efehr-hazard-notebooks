

# Hazard Notebooks

This repository contains interactive notebooks for exploring earthquake hazard data in a simple and practical way.

The main goal is to help new users understand what is inside seismic hazard datasets and how different model choices affect results.

## What this notebook does

The notebook lets you:
- browse available source model files
- choose one or more files to analyze
- look at where earthquake sources are located on a map
- explore how earthquake rates change with magnitude
- compare different model versions in a simple and visual way

The focus is not on advanced theory, but on helping you build intuition by seeing the data directly.

## How to use it

1. Open the notebook
2. Run the cells from top to bottom
3. When you reach the file selection step, choose the files you want to explore
4. Continue running the notebook to see maps and statistics

You can experiment by changing the selected files and rerunning the notebook.

## Who this is for

- new team members learning hazard workflows
- researchers who want a quick look at model differences
- anyone curious about how seismic source models behave

## Notes

- The notebook works directly with online data
- You do not need to download the datasets manually
- Start simple (one or two files), then try more comparisons
